#scala_bigdata
本项目涵盖了 Hbase、Spark、Flink 的 scala 样例代码
###安装要求
    二进制包版本
    - jdk：1.8
    - scala：2.11.12
    - spark：2.1.1
    - flink：1.9
    - hbase：1.2.0
    - hadoop：2.7.5
    - kafka：2.10-0.10.2.2
###安装步骤
    git clone https://github.com/apache/flink.git
    cd dir
    mvn clean package -DskipTests
###版本控制
该项目使用 Git 进行版本管理。
